## Resource Hacker 资源修改工具汉化版
Resource Hacker 是一款功能强大且易用的工具，主要用于查看和编辑 Windows PE 文件(exe、dll等类型)中的资源。

这里是 Resource Hacker 的汉化版，仅是做了汉化，无任何病毒、木马、后门。




### 汉化修改内容
1、将界面和提示内容进行汉化。

2、优化部分选择文件的界面，增加“所有文件”选项，这样不用专门修改成对应的后缀名。
